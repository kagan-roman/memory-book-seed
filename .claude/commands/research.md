Стадия **research**. Прочитай `book.config.yml`, `CLAUDE.md` (§5 опечатки, §6
архивы, §11 глубина).

Для каждого человека (или для `$ARGUMENTS`, если задан slug) запусти агента
`researcher`. Он пишет `book/chapters/<slug>/facts.md` + `context.md`.

- Батчами по `run.parallel_batch` (3–4), по одному человеку на агента.
- chrome-devtools MCP для pamyat-naroda; сметка всех `surname_variants`;
  глубина по `run.depth` («прокликать каждую страницу» на exhaustive).
- Если агент сообщает про капчу/гео-фенс — передай пользователю просьбу
  открыть ссылку у себя и вставить (не блокируй весь конвейер).
- **Проверь вывод** каждого researcher на состыковку, прежде чем считать
  готовым (§11): не приписана ли чужая часть, не задвоены ли награды.

После прохода — короткая сводка: по кому богатый материал, по кому пробелы,
что осталось докопать. Ничего не выдумывать; «источников не найдено» — норма.

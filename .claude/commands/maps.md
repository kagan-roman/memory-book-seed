Стадия **maps**. Прочитай `book.config.yml`, `CLAUDE.md` (§8), `build/RULES.md`.

Проверь, что геоданные есть (`assets/geo/clipped_*.geojson`). Нет — сначала
`python3 build/clip_geo.py` (скачивает Natural Earth, режет по `geo:`).

Для каждого человека (или `$ARGUMENTS`) запусти агента `cartographer`:
`book/chapters/<slug>/maps.json` → `python3 build/build_maps.py --slug <slug>`
→ `build/.out/maps/<slug>_NN.html`+`.png`. Батчами по `run.parallel_batch`.

Жёстко: **линию фронта не рисуем** (§8); дом/призыв/село не на боевую карту;
далёкая точка (учебка в тылу) — в био-панель, не растягивать масштаб; города
из текста — подписать, старое(новое) имя; halo перекрывает фон полностью.
Мало данных — карты нет (честно), либо карта-якорь (STYLE-RULES §5.2).

Проверь PNG глазами: нет наезжающих подписей, нет линии фронта, нет фейкового
моря по краям. Сводка: у кого сколько листов, где приблизительные точки, у
кого карты нет и почему.

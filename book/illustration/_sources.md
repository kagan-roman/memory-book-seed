# Каталог фотоархивов (переиспользуемый)

Ведёт и пополняет `illustrator`. Что держит источник, по каким фронтам/
операциям/частям, как искать, способ доступа, режим прав. Сверяться **первым**,
дополнять по ходу. Только документальные фото мест/операций (не документы, не
портреты, без AI — `CLAUDE.md §13–§15`).

| Источник | Что держит | Доступ | Права |
|---|---|---|---|
| goskatalog.ru | Госкаталог Музейного фонда РФ — атрибуция + статус прав | гео-фенс РФ → попросить пользователя открыть и вставить | по карточке (часто PD/музей) |
| russiainphoto.ru / МАММ | советская фотохроника | WebFetch / браузер | по карточке |
| waralbum.ru | фронтовое фото с указанным первоисточником | WebFetch | по первоисточнику |
| Фотофонды ЦАМО, ТАСС | боевые кадры по номеру части | через pamyat-naroda / музеи | архив |
| Wikimedia Commons | PD-карты Генштаба, PD-Russia | WebFetch | PD (проверять срок) |
| Bundesarchiv | PD-кадры противоположной стороны | WebFetch | PD (узко, точная атрибуция) |

> Англоязычный веб (Getty/Alamy/Wikipedia) — **не первоисточник**: лишь след
> к настоящему архиву. Ищи по-русски (место/операция/часть/год).

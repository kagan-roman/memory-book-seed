# Чек-лист иллюстраций — ГЕЙТ АВТОРА

`illustrator` (фаза PROPOSE) заполняет таблицу кандидатами (3–5 на главу,
тиры A/B/C, тест атрибуции 5 пунктов). **Вы** проставляете в колонке `✔`:
`да` / `нет` / `другой кадр: …`. Пусто = не вставлять. Затем
`/illustrate apply`.
